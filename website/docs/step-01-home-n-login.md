## 5. Create Home and Login Page

We don't have Home and Login page yet. Let's create them.

`src/pages/Home.jsx`
```
import React, { Component } from 'react';

export default class Home extends Component {
  render() {
    return (
      <h1>Home</h1>
    )
  }
}
```

`src/pages/Login.jsx`
```
import React, { Component } from 'react';

export default class Home extends Component {
  render() {
    return (
      <h1>Login</h1>
    )
  }
}
```
